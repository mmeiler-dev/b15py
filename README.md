# b15py

Ein Module um das B15F-Board mit Python zu programmieren
Dies ist ein Python Module für den B15F-Treiber.

> [!WARNING] 
> Für die Verwendung dieses Modules muss man die nötigen Header Files installiert haben
> Für Arch-Nutzer kann man dies einfach über einen AUR-Helper machen.
> Bsp.: `yay b15f-git`

## Installation

Gerade ist dies nur ein Git-Repo, da ich ein wenig mit PyPI kämpfe.
Einfach `git clone https://github.com/mmeiler-dev/b15py` und dann `pip install .` im Terminal verwenden.
(Später dann einfach mit `pip install b15py` hoffentlich)
